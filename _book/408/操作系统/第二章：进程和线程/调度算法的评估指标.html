调度算法的评估指标

调度算法的评估指标

   

  • CPU 利用率
    • 利用率=CPU工作的时间总时间\small \text{利用率} = \dfrac{\text{CPU工作的时间}}{\text{总时间}}利用率=​总时间​​CPU工作的时间​​
    • 有些题目会让我们计算其他设备的利用率，同理。
  • 系统吞吐量
    • 系统吞吐量=总共完成了多少道作业总共花了多少时间\small \text{系统吞吐量} = \dfrac{\text{总共完成了多少道作业}}{\text{总共花了多少时间}}系统吞吐量=​总共花了多少时间​​总共完成了多少道作业​​
    • 即计算单位时间内能完成多少道作业。
  • 周转时间
    • 周转时间=作业完成时间−作业提交时间\small \text{周转时间} = \text{作业完成时间} - \text{作业提交时间}周转时间=作业完成时间−作业提交时间
    • 平均周转时间=每一个作业的周转时间之和作业的个数\small \text{平均周转时间} = \dfrac{\text{每一个作业的周转时间之和}}{\text{作业的个数}}平均周转时间=​作业的个数​​每一个作业的周转时间之和​​
    • 带权周转时间=作业的周转时间作业实际运行的时间\small \text{带权周转时间} = \dfrac{\text{作业的周转时间}}{\text{作业实际运行的时间}}带权周转时间=​作业实际运行的时间​​作业的周转时间​​
    • 平均带权周转时间=各作业的带权周转时间之和作业数量\small \text{平均带权周转时间} = \dfrac{\text{各作业的带权周转时间之和}}{\text{作业数量}}平均带权周转时间=​作业数量​​各作业的带权周转时间之和​​
  • 等待时间
    • 进程/线程等待被服务的时间之和（即周转时间减去 CPU 为其服务的时间再减去 I/O 设备时间）。
    • 平均等待时间：每一个进程/线程等待时间的平均值。
  • 响应时间
    • 从用户提交到首次响应之间的时间。
