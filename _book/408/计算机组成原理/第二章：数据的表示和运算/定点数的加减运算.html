无符号数的加减法

无符号数的加法很简单, 我们重点关注一下减法, 假设我们要计算 A−BA - BA−B

我们应该对 BBB 按位取反得到 CCC 之后将 A−BA-BA−B 转化位 A+C+1A + C + 1A+C+1

原理如下: 假设我们操作数是有 kkk 位的, 那么我们能得出 B+C=2k−1B + C = 2^k - 1B+C=2​k​​−1, 所以有 B+C+1=2kB + C + 1 = 2^kB+C+1=2​k​​

所以 −B=C+1−2k-B = C + 1 - 2^k−B=C+1−2​k​​ , 所以 A−B=A+C+1−2kA - B = A + C + 1 - 2^kA−B=A+C+1−2​k​​ , 因为一共就 kkk 位, 所以 2k2 ^k2​k​​ 会给模运算忽略掉, 所以 A−B=A+C+1A- B = A + C + 1A−B=A+C+1

所以无符号加减法的电路和有符号加减大的电路一样
